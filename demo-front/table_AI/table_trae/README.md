# 黑八桌球游戏 - 移动端优先实现

一个基于HTML5 Canvas和JavaScript开发的黑八桌球游戏，特别优化了移动端触摸体验。

## 游戏简介
这是一款经典的黑八桌球游戏，支持触摸操作，玩家可以在手机或平板上享受真实的台球体验。游戏包含完整的物理碰撞系统、球体运动模拟和游戏规则实现。

## 技术栈
- HTML5 Canvas - 用于游戏渲染
- JavaScript - 游戏逻辑和物理引擎
- Tailwind CSS v3 - 样式和UI组件
- Font Awesome - 图标

## 游戏规则
1. 游戏开始时，玩家通过点击白球并拖动来控制击球方向和力度
2. 先打进一组球（全色或花色）的玩家需要最后打进黑八球获胜
3. 如果在打进自己的球组前打进黑八球，将输掉比赛
4. 白球入袋或飞出球桌会导致自由球
5. 当一方打进所有自己的球后，打进黑八球即可获胜

## 移动端操作说明
- **触摸控制**：点击白球并拖动来瞄准，拖动距离决定击球力度
- **角度调整**：拖动时可以调整击球角度
- **击球**：松开手指即可击球

## 如何运行游戏

### 方法1：直接打开HTML文件
由于这是一个纯前端项目，你可以直接在浏览器中打开`index.html`文件来运行游戏。

### 方法2：使用本地服务器
对于更好的体验，建议使用本地服务器运行：

```bash
# 使用Python的内置服务器
python -m http.server

# 或者使用Node.js的http-server
npx http-server
```

然后在浏览器中访问提示的URL地址。

## 项目特点
- **移动端优先设计**：完全适配触摸操作和移动设备屏幕
- **真实的物理引擎**：模拟台球的碰撞、反弹和摩擦效果
- **响应式布局**：自动适应不同屏幕尺寸
- **精美的视觉效果**：包括球的纹理、阴影和高光
- **完整的游戏规则**：实现了标准黑八桌球的全部规则

## 游戏控制
- **重新开始**：点击右上角的刷新图标
- **游戏规则**：点击右上角的问号图标查看规则
- **触摸白球**：开始瞄准和击球
