# 🎱 黑八桌球游戏

一个使用HTML5 Canvas和JavaScript开发的经典黑八桌球游戏。

## 🎮 游戏特色

- **真实物理引擎**：包含摩擦力、碰撞检测和反弹效果
- **精美界面**：现代化的UI设计，流畅的动画效果
- **完整规则**：遵循标准黑八桌球游戏规则
- **响应式控制**：鼠标控制，直观易用

## 🚀 如何开始

1. 在浏览器中打开 `index.html` 文件
2. 游戏将自动加载并开始

## 🎯 游戏规则

### 基本目标
将8号球（黑球）击入袋中，但必须在最后击入。

### 详细规则
1. **击球顺序**：必须先击打自己的球（花色球或实色球）
2. **犯规情况**：
   - 白球入袋
   - 8号球过早入袋
   - 未击中任何球
3. **胜利条件**：将所有球击入袋中，最后击入8号球

## 🖱️ 操作方法

- **瞄准**：移动鼠标瞄准目标
- **蓄力**：按住鼠标左键蓄力
- **击球**：释放鼠标左键击球
- **重新开始**：点击"重新开始"按钮

## 🛠️ 技术实现

- **HTML5 Canvas**：游戏渲染
- **JavaScript ES6**：游戏逻辑和物理引擎
- **CSS3**：界面样式和动画
- **物理引擎**：自定义实现的碰撞检测和运动学

- 游戏状态显示

## 🔧 自定义设置

可以通过修改 `game.js` 中的参数来调整游戏：

- `friction`：摩擦力系数
- `maxPower`：最大击球力度
- `ballRadius`：球的大小
- `canvas.width/height`：游戏区域大小
