# 游戏规则面板更新

## 📋 更新概述

为了帮助玩家更好地理解和使用新增的球杆显示和轨迹预测功能，我们对游戏规则面板进行了全面更新。

## ✅ 新增内容

### 1. 辅助功能说明

新增了专门的"🎱 辅助功能"部分，详细介绍：

- **球杆显示**: 瞄准时自动显示球杆位置
- **轨迹预测**: 白色虚线显示白球运动路径
- **碰撞预测**: 橙色虚线显示目标球轨迹
- **碰撞点**: 红色圆点标记预期碰撞位置
- **力度反馈**: 球杆距离反映击球力度

### 2. 轨迹说明详解

新增了"🎯 轨迹说明"部分，包含：

- **颜色编码**: 不同颜色线条的含义
- **视觉元素**: 各种标记的作用
- **物理特性**: 反弹和摩擦力的考虑
- **视觉效果**: 透明度渐变的说明

### 3. 游戏技巧增强

在原有技巧基础上，新增了轨迹预测相关的高级技巧：

- **利用轨迹预测**: 观察白色虚线规划路线
- **注意碰撞点**: 红点显示最佳击球位置
- **预判目标球**: 橙色虚线显示目标球去向
- **调整球杆距离**: 距离越远力度越大

## 🎨 视觉设计

### 颜色标识

在规则说明中使用了对应的颜色来标识不同元素：

- `text-white`: 白色虚线（白球轨迹）
- `text-orange-400`: 橙色虚线（目标球轨迹）
- `text-red-400`: 红色圆点（碰撞点）
- `text-red-300`: 红色光环（碰撞强调）
