# 黑八桌球 - Kiro AI

一个基于 Vue 3 + Canvas 2D 开发的黑八桌球游戏，专为移动端优化设计。

## 🎯 项目特色

- **2D渲染**: 使用 Canvas 2D 提供流畅的游戏体验
- **移动端优先**: 专为触摸设备优化，竖屏布局
- **物理引擎**: 自研2D物理引擎，真实的球体运动
- **触摸友好**: 直观的触摸操作和手势控制
- **响应式设计**: 自适应不同屏幕尺寸
- **现代技术栈**: Vue 3 + TypeScript + Vite

## 🎱 游戏规则

### 基本规则
1. **开球**: 游戏开始时进行开球，第一个进球决定玩家的球组
2. **球组分配**:
   - 实心球组 (1-7号球)
   - 花色球组 (9-15号球)
3. **获胜条件**: 先打完自己球组的所有球，再合法击入黑8球获胜

### 操作方法
- **移动端**: 触摸白球并拖动瞄准，拖动距离决定击球力度
- **球杆显示**: 瞄准时自动显示球杆，直观显示击球方向
- **轨迹预测**: 实时显示白球和目标球的运动轨迹
- **碰撞预测**: 高亮显示预期碰撞点和碰撞后轨迹
- **力度控制**: 底部滑块可精确调整击球力度
- **暂停/继续**: 点击暂停按钮
- **游戏规则**: 点击❓按钮查看详细规则和辅助功能说明

### 犯规情况
- 白球进袋
- 击打对方球组的球
- 过早击打黑8球
- 击球后没有球碰到边框

## 🏗️ 技术架构

### 前端框架
- **Vue 3**: 采用 Composition API 和 TypeScript
- **Vite**: 现代化构建工具，支持热重载
- **Tailwind CSS**: 原子化CSS框架

### 2D渲染
- **Canvas 2D**: 高性能2D图形渲染
- **自适应**: 响应式画布尺寸

### 物理引擎
- **自研2D物理**: 轻量级物理模拟
- **碰撞检测**: 精确的球体碰撞
- **边界反弹**: 真实的桌边反弹效果

### 移动端优化
- **触摸优先**: 专为触摸设备设计
- **竖屏布局**: 符合手机使用习惯
- **性能优化**: 针对移动设备优化渲染
